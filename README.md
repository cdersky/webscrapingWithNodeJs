#Install


1.clone the repo
```$ git clone git@github.com:cdersky/webscrapingWithNodeJs.git```

2.switch to new directory
```$ cd webscrapingWithNodeJs```

3.install dependencies
```$ npm install```

4.run an examples
```$ npm run example1```
